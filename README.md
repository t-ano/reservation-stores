### 利用した技術

- Laravel 8　（PHPフレームワーク）
- Laravel Breeze　（認証機能パッケージ）
- SQLite　（データベース）
- Tailwind CSS　（CSSフレームワーク）

### **要件**

- ユーザーの予約を受け付けることができる。
- 予約時に日付、予約プランで絞り込みができる。
- 複数店舗の管理ができる。
- 売上合計の日々、月間、年間を表示できる。
- 顧客データの管理ができる。
- 予約時にクレジットカード決済が可能。
- 売上データのCSVダウンロードが可能。
- 管理者IDとPASSが発行できログインできる。

### 機能

**管理者側**
- 認証機能
- 店舗の登録、編集、削除
- 予約プランの登録、編集、削除
- 顧客データの一覧表示
- 売上を日々、月間、年間で表示

**利用者側**
- 日付、予約プランの絞り込み
- 予約機能
- 利用者情報の入力
- 決済機能
